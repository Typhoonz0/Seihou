<h2>Controls</h2>

<h3>Player 1</h3>
	<ul>
		<li>W: Jump</li>
		<li>A: Left</li>
		<li>D: Right</li>
		<li>T: Punch</li>
		<li>Y: Kick</li>
	</ul>

<h3>Player 2</h3>
	<ul>
		<li>Up Arrow: Jump</li>
		<li>Left Arrow: Left</li>
		<li>Right Arrow: Right</li>
		<li>O: Punch</li>
		<li>P: Kick</li>
	</ul>

<h2>Setup and Download</h2>
	<p><b>I recommend Python 3.11 as you will need extra tooling to compile Pygame from source on the current version of Python.</b></p>
```bash
	git clone https://github.com/Typhoonz0/Seihou.git 
	cd Seihou 
	pip install -r requirements.txt
```
