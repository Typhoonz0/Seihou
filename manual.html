Manual

Seihou is a simple fighting game.

Controls

Player 1

  • W: Jump
  • A: Left
  • D: Right
  • T: Punch
  • Y: Kick

Player 2

  • Up Arrow: Jump
  • Left Arrow: Left
  • Right Arrow: Right
  • O: Punch
  • P: Kick

Setup and Download

I recommend Python 3.11 as you will need extra tooling to compile Pygame from source on the current version of Python.

git clone https://github.com/Typhoonz0/Seihou.git
cd Seihou
pip install -r requirements.txt
